# Variation Sets extractor from CHA files

It uses [CHA file parser for Python](https://github.com/macramole/chafile)

## Features
Configurable:
* Speaker interventions
* Child interventions
* Other speaker to child interventions 
* Other speaker among adults interventions
* Max time
